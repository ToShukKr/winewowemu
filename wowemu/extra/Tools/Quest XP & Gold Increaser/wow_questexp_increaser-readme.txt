With this script you can increase the exp_rate of all the quests
After you ran the script (go.bat), you will see 2 files
1: quests_old.scp -> the old quests, unchanged
2: quests.scp -> the new quests, changed with the new reward_xp rate


When you click on go.bat, windows will popup, and then type there the multipliyer
like if you want to increase xp rate by 25 times, then type 25..


NOTE: This will change ALL the reward_xp rates!
